The Student Activity Portal 2026 is a fully responsive, Bootstrap 5-based web portal designed for FAST-NU Chiniot-Faisalabad Campus students. It allows students to explore campus events, register for activities, and connect with student societies. The project demonstrates mastery of Bootstrap's core layout system, components, and utility classes — including responsive grids, navbar, modals, badges, alerts, typography, and spacing utilities — across 10 structured tasks.
